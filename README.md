🦸‍♂️ Marvel Interactive Map
Este proyecto es una experiencia web interactiva que permite explorar a los personajes más icónicos del Universo Marvel a través de una imagen mapeada.

🚀 Características
Imagen Interactiva: He utilizado un mapeado de coordenadas para que los personajes principales reaccionen al pasar el ratón.

Efecto Hover & Enlaces: Al posicionar el cursor sobre un héroe, se activa un estado de selección y, al hacer clic, serás redirigido automáticamente a su biografía oficial en Wikipedia.

Guía de Estilo Visual: En la parte inferior, encontrarás una lista detallada de los personajes, cada uno resaltado con su color característico para una identificación rápida.

🛠️ Tecnologías utilizadas
HTML5: Estructura y mapeado de imagen (<map> y <area>).

CSS: Estilos personalizados, efectos de hover y diseño de la lista de personajes.
